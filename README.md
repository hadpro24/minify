# minify

## Authorization Server

```bash
cd iam

pip install -r requirements.txt

python manage.py runserver 0:5000
```

## Ressource Server

```bash
cd api

pip install -r requirements.txt

python manage.py runserver 0:7000
```

## Application Client

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
